# Warm Start - MMS

This repository contains the code for the warm start method for the Mission Management System (MMS), developed at Centre Internacionals de Mètodes Numèrics en Enginyeria (CIMNE), Barcelona, Spain.

## Installation

Create a new environment (if needed):
```bash
python -m venv .venv
source .venv/bin/activate
```

If `python -m venv .venv` fails, try `python3 -m venv .venv`.

Install the dependencies in an existing environment:
```bash
pip install -r requirements.txt
```

## Run

Run the main script `astar_finder.py`.
